# 🚀 Phase 3 快速測試指南

> 5 分鐘快速驗證所有 AI 功能

## ⚡ 快速測試清單

### 1️⃣ 語義搜尋 (1 分鐘)

**位置**: Dashboard 或首頁搜尋框

**測試**:
```
✓ 輸入: "wireless headphones"
✓ 應該顯示: Premium Wireless Headphones
✓ 檢查: 有相似度評分和 AI 標記
```

---

### 2️⃣ AI 聊天 (2 分鐘)

**位置**: 右下角紫色懸浮按鈕

**測試對話**:
```
你: "What products do you have?"
✓ AI 應該: 即時串流回應，帶打字效果

你: "I need a tech gift under $200"
✓ AI 應該: 推薦相關商品

你: "Tell me more about the first one"
✓ AI 應該: 記住前一個商品（上下文）
```

---

### 3️⃣ 個人化推薦 (30 秒)

**位置**: Dashboard 的推薦區塊

**檢查**:
```
✓ 顯示 4 張商品卡片
✓ 有交錯淡入動畫
✓ 標記為 "Personalized" 或 "Popular"
✓ Hover 時卡片有陰影效果
```

---

### 4️⃣ AI Demo 頁面 (1 分鐘)

**位置**: `/ai-demo`

**檢查**:
```
✓ 三個功能區塊都顯示
✓ 搜尋框可以使用
✓ 推薦商品正常顯示
✓ Phase 3 功能列表完整
```

---

### 5️⃣ 效能快檢 (30 秒)

**測試項目**:
```
✓ 搜尋結果: < 1 秒
✓ AI 首字回應: < 0.5 秒
✓ 頁面載入: < 3 秒
✓ 動畫流暢，無卡頓
```

## 🐛 發現問題？

### 常見問題速查

**問題**: 搜尋沒結果
**解決**: 確認商品已向量化
```bash
npm run check-products
```

**問題**: AI 聊天無回應
**解決**: 檢查是否已登入

**問題**: 推薦不顯示
**解決**: 確認已登入並有瀏覽歷史

**問題**: 速率限制錯誤
**解決**: 等待 15 分鐘重置（AI: 20次/15分鐘）
